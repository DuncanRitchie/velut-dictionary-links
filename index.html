Duncan’s Affix Concatenator for velut







About

velut is my Latin vocabulary website. When adding words to velut, I sometimes find I want every possible combination from concatenating one set of text strings (which I’ll call prefixes) onto another set of text strings (suffixes). This page helps me do this.

If you like, you can load a small amount of sample data using the button at the top of the page, so you can see what this page does. Or you can type or paste your own data in.

If the “Output the second column?” tickbox is ticked, each concatenation will be output next to a Tab character and the first concatenation for the prefix.
