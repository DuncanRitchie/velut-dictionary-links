Duncan’s Links to Online Dictionaries for velut

About

velut is my Latin vocabulary website. When adding words to velut, I find it useful to check them in online dictionaries. This page helps me do this.

To use it, enter a load of words into the textarea, and select from a list of URL templates. Then click a link. The href of the link comes from an array of strings created by combining the words with the URL templates. Clicking the link makes it cycle through the array.

If you like, you can load a small amount of sample data using the button at the top of the page, so you can see what this page does. Or you can type or paste your own data in.
